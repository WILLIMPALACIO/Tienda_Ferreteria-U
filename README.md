🛠️ Sistema de Gestión Ferretería - Aplicación Web

Aplicación web desarrollada para la gestión y visualización de productos de una ferretería. Permite explorar herramientas, consultar detalles de productos, realizar búsquedas dinámicas y gestionar un carrito de compras de forma intuitiva.
