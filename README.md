🛠️ Sistema de Gestión Ferretería - Aplicación Web

Aplicación web desarrollada para la gestión y visualización de productos de una ferretería. Permite explorar herramientas, consultar detalles de productos, realizar búsquedas dinámicas y gestionar un carrito de compras de forma intuitiva.

<img src="https://skillicons.dev/icons?i=angular,ts,html,css,bootstrap" />

✨ Funcionalidades

- Catálogo de productos.
- Visualización de detalles de cada producto.
- Búsqueda de herramientas en tiempo real.
- Carrito de compras.
- Interfaz responsiva con Bootstrap.
- Navegación entre secciones mediante Angular Router.
